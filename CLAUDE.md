# CLAUDE.md — v2t

Краткий контекст для Claude / Cursor в этом репозитории.

## Что это

Портативная утилита **медиа → текст**: локальные файлы, папки, ссылки (в т.ч. плейлисты). Сборка под **Windows** и **macOS**.

## Стек

- **Tauri 2** + **React** + **TypeScript** + **Vite**
- **ffmpeg**, **yt-dlp** — рядом с приложением или пути в настройках
- Транскрипция: HTTP API (Whisper-совместимый или другой провайдер из настроек)

## Структура (после scaffold)

- `src/` — фронтенд
- `src-tauri/` — Rust, `tauri.conf.json`, capabilities
- Бинарники sidecar — по документации Tauri для целевой ОС

## Порты

Десктоп: dev-сервер Vite (часто `1420` / `5173` — смотри `vite.config` и Tauri).  
Если появится Docker или отдельный backend — **обязательно**: глобальный реестр `C:\Users\vetal\PORTS.md` и правила в `C:\Users\vetal\.cursor\rules\ports-global.mdc`.

## Документация для агентов

- **`AGENTS.md`** — роли, субагенты, поведение
- **`docs/PLAN.md`** — поэтапный план реализации (Фазы 1–6)
- **`.cursor/tasks/BA-20250322-120000-alice.md`** — BA-анализ (Алиса) для оркестратора
- **`.cursor/rules/v2t-project.mdc`** — правила репозитория

## Безопасность

- Не хранить API-ключи в коде
- Безопасный запуск внешних процессов (аргументы, не shell из пользовательского ввода)
